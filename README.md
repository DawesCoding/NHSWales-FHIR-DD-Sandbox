# NHS Wales FHIR Profiles - R4
This repository is a sandbox environment used by <@DawesCoding> to test changes to the NHS Wales package and associated Implementation Guide assets, primarily to assure the impact of structural changes on functionality.
This repository is not an official NHS Wales release, and should not be used for development.
Please refer to <https://simplifier.net/guide/fhir-standards-wales-implementation-guide> for official NHS Wales contents.
